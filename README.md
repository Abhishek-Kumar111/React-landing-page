# React Landing Page 

A modern, responsive landing page built using **React.js** and **Tailwind CSS**.  
This project is developed as part of an academic assignment to demonstrate frontend development skills.

---

##  Project Objective

The objective of this project is to design and develop a clean and responsive landing page using React.js with a component-based architecture and Tailwind CSS for styling.

---

##  Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6)
- HTML5

---

##  Features

- Responsive design (mobile, tablet, desktop)
- Component-based structure
- Modern UI using Tailwind CSS
- Fast performance
- Deployed on Vercel (Free)
